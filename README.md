# ✅ To-Do List App

Heyy! I’m Spoorthy S G.  
This is a simple and interactive **To-Do List App** built using **HTML, CSS, and JavaScript**.  
It helps you **organize tasks, track progress, and boost productivity** with a clean user interface.  

---

## 🔥 Features
- Add, edit, and delete tasks 📝  
- Mark tasks as completed ✔️  
- Stores tasks in **local storage** (so your list stays even after refreshing) 💾  
- Responsive and beginner-friendly UI 🎨  

---

## 🛠️ Technologies Used
- **HTML** – Structure  
- **CSS** – Styling  
- **JavaScript** – Functionality
